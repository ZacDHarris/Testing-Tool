CGNAT Reference
Identification: Private IP
  • To confirm CGNAT, review the customer's router IP. If it falls in a private range, the customer is not on a public IP.
  • 100.64.0.0 – 100.127.255.255
  • 10.0.0.0 – 10.255.255.255
  • Example: Router IP 100.115.26.47 → Indicates private IP under CGNAT
Common CGNAT Symptoms
  • Inability to host/join online multiplayer games
  • Non-functional port forwarding despite correct setup
  • Cannot access home devices remotely (IP cameras, smart home)
  • VPN services not connecting or inconsistent
  • Router IP is in a private range (100.x.x.x or 10.x.x.x)
Escalation
  • If one or more symptoms are reported and a private IP is confirmed, escalate as outlined.
